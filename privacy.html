隐私政策

最后更新日期：2026年5月10日

1. 信息收集

FinView 是一个纯前端网站，我们不会在服务器上收集或存储您的个人身份信息。以下信息仅存储在您的本地浏览器中：

  • 自选列表：您收藏的交易品种偏好，存储在浏览器 localStorage 中
  • 主题偏好：您的深色/浅色模式选择，存储在浏览器 localStorage 中

2. 第三方 API

本网站使用以下第三方服务获取市场数据：

  • ExchangeRate API — 外汇汇率数据
  • CoinGecko API — 加密货币价格数据
  • RSS2JSON — 财经新闻聚合

这些第三方服务可能有各自的隐私政策，建议您查阅其官方网站了解详情。

3. Cookies

本网站不使用传统 cookies。所有用户偏好设置均通过浏览器 localStorage 存储，您可以随时在浏览器设置中清除这些数据。

4. 数据安全

由于我们不在服务器端存储任何用户数据，因此不存在服务器端数据泄露风险。您的自选列表和主题偏好仅存在于您的本地设备中。

5. 免责声明

本网站提供的所有市场数据和资讯仅供参考，不构成任何投资建议。用户基于本网站信息进行的任何交易决策，风险由用户自行承担。

6. 联系我们

如果您对本隐私政策有任何疑问，请通过网站底部的联系方式与我们取得联系。
